# Amravati Municipal Corporation Property Tax Dashboard

## Project Overview

Designed an interactive Power BI dashboard for analyzing property tax collection, pending payments, and revenue trends for Amravati Municipal Corporation.

## Objectives

* Monitor property tax collection performance
* Analyze pending and paid taxes
* Track zone-wise and category-wise revenue
* Improve tax reporting and visualization

## Tools & Technologies

* Power BI
* SQL
* Excel

## Key Features

* Tax Collection Dashboard
* Zone-wise Revenue Analysis
* Property Category Analysis
* Outstanding Tax Monitoring
* Interactive Reports and Filters

## Business Insights

* Identified zones with low tax collection
* Tracked pending payments and revenue trends
* Improved reporting efficiency through visualization
